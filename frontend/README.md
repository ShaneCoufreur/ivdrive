# iVDrive Frontend

Part of [iVDrive](https://github.com/m7xlab/iVDrive) — see the repository root [README](../README.md) for project overview, architecture, and getting started.

## Local development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000). The app uses the same-origin `/api/*` proxy by default; set `NEXT_PUBLIC_API_URL` if the API runs elsewhere.

## Build

```bash
npm run build
```

## Tech stack

Next.js (App Router), React, Tailwind CSS, Recharts, React-Leaflet, Lucide React.
