# Thank You

iVDrive would not exist without the inspiration and work of these projects and communities. Thank you.

## MySkoda

**[MySkoda](https://myskoda.readthedocs.io/en/stable/)** · [skodaconnect/myskoda](https://github.com/skodaconnect/myskoda)

Async Python library to interact with the Škoda API. MySkoda powers how we talk to Škoda Connect and inspired our approach to vehicle data.

## CarConnectivity

**[CarConnectivity](https://github.com/tillsteinbach/CarConnectivity)** · [Till Steinbach](https://github.com/tillsteinbach)

Python API for telemetry from various car brands, with a connector and plugin architecture. It showed what’s possible when building on top of brand APIs and influenced how we think about EV data.

## TeslaMate

**[TeslaMate](https://github.com/teslamate-org/teslamate)** · [teslamate-org](https://github.com/teslamate-org)

Self-hosted data logger for Tesla with dashboards and a clear user focus. It inspired the idea of a dedicated, user-focused EV data experience for other brands.

---

These projects inspired us to build **iVDrive** — focused on **users** first: a simple way to use your Škoda (and future VW Group) EV data online or self-hosted, without needing to be a developer.

Thank you to the maintainers and contributors of these projects.
