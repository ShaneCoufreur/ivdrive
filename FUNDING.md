# Support iVDrive

iVDrive is developed and run as a service at [ivdrive.eu](https://ivdrive.eu) so that anyone can use it without setting up servers. To keep the service and domain running, we rely on voluntary support from the community.

## Where your support goes

- **Domain** (ivdrive.eu) and **hosting** for the public instance
- **Target:** ~€200/month to cover costs and keep the project sustainable

Every contribution helps, whether one-time or recurring. Thank you for considering it.

## How to support

Use the **Sponsor** button at the top of this repository on GitHub — it shows the available options (e.g. GitHub Sponsors, Ko-fi, PayPal, or other links we’ve configured).

If you prefer not to use the button, the same links are listed in the repository’s **About** section (when configured).

---

**Maintainers:** To show the Sponsor button and add your donation links (PayPal, Ko-fi, GitHub Sponsors, etc.), edit [.github/FUNDING.yml](.github/FUNDING.yml). Add at least one active link (e.g. `custom: ['https://www.paypal.me/yourpage']` or `ko_fi: yourusername`). See [GitHub’s funding docs](https://docs.github.com/en/repositories/managing-your-repositorys-settings-and-features/customizing-your-repository/displaying-a-sponsor-button-in-your-repository).

- **Contribute** — Code, docs, and feedback: see [CONTRIBUTING.md](CONTRIBUTING.md)
