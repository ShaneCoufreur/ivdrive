# iVDrive — Logo & Brand Identity Guide

## Overview

**iVDrive** is a premium electric vehicle data monitoring application for Volkswagen Group EVs,
starting with Škoda electric vehicles via the `mysmob.api.connect.skoda-auto.cz` API.
The application collects, visualises, and presents real-time EV telemetry data — battery,
range, charging state, location and more — in a sleek, driver-friendly interface.

---

## Brand Concept

The logo fuses two worlds:
- ⚡ **Tesla-style minimalism** — dark background, bold monogram, premium tech feel
- 🟢 **Škoda iV electric identity** — emerald green, aerodynamic lines, European EV sophistication

The result is a confident, modern icon that communicates *intelligence, electric power, and connectivity*.

---

## Logo Mark: The `iV` Monogram

### Symbol Elements

| Element | Description |
|---|---|
| **`i`** | Lowercase, slim letterform. The dot is replaced by a ⚡ lightning bolt, symbolising electric power and the EV connection |
| **`V`** | Bold, angular. The strokes echo the roofline silhouette of a modern EV hatchback (e.g. Škoda Enyaq). The upper edge carries a subtle aerodynamic sweep/wing detail |
| **Ring** | A circular gradient ring frames the monogram — referencing a charging ring / battery indicator arc |

### Symbolism
- ⚡ Lightning bolt = electric vehicle / EV charging
- The `V` sweep = speed, motion, aerodynamics
- Circle ring = charging state / battery loop
- `iV` = Volkswagen Group's electric vehicle suffix (e.g. Škoda **iV** lineup: Enyaq iV, Octavia iV)

## Usage Guidelines

### ✅ Do
- Use on dark backgrounds for maximum impact
- Maintain the circular ring framing when displaying the icon
- Keep the gradient direction consistent (green bottom-left → cyan top-right)
- Use `Inter` or `Outfit` for any accompanying text

### ❌ Don't
- Stretch or distort the `iV` monogram
- Use on busy photographic backgrounds without a dark overlay
- Replace the lightning bolt dot with a plain circle dot
- Use low-contrast color combinations that reduce legibility
