# iVDrive documentation

Documentation for using and running iVDrive.

## For users

- **Use iVDrive online** — Go to [ivdrive.eu](https://ivdrive.eu), register, add your Škoda Connect vehicle, and view your data. No setup required.
- **Self-hosting** — See [Self-hosting guide](self-hosting.md) if you want to run the stack yourself (Docker).
